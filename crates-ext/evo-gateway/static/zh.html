功能说明 ▼
智能体运行时
  • 自主进化 YAML 技能树（EWMA 生命周期）
  • 每次任务后自动反思与技能提炼
  • 仅追加 JSONL 审计日志 + 会话重放
  • 6 层工作记忆（L0–L5）
  • 5 重 Token 节省策略（约为原始成本⅓）
安全与隐私
  • 密钥保险库（chmod 600）——永不发送给模型
  • 模式识别脱敏：sk-*、ghp_*、JWT、高熵串
  • credentials.toml——文本编辑器直接编辑
  • 凭证在 Rust 执行边界注入，模型只见占位符
  • TOTP 本地计算（HMAC-SHA1）——种子不离本机
  • policy.toml——按工具配置允许/拒绝规则 + 执行前钩子
浏览器自动化
  • 无头 Chrome：导航、点击、输入、截图、执行 JS
  • 持久会话——首次登录后长期复用
  • 懒登录检测——会话有效时直接跳过认证
  • 账户隔离 Chrome Profile（chmod 700）
  • 全自动 TOTP · 半自动短信码 · 无 2FA 自动跳过
集成与接入
  • 17 家模型厂商（9 国内 + 8 国际 + 3 本地）
  • ACP — Claude Code、Codex、Cursor、Copilot…
  • MCP — GitHub、文件系统、Postgres、Brave…
  • 消息渠道 — Telegram、Slack、Discord
  • 本地 HTTP 网关（即此页面）
